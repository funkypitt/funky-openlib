# OpenLibeExtended-eInk-Remix

**An open-source e-ink optimized e-book reader with library search capabilities.**
**Optimized for e-paper displays with paginated reading, adaptive font sizing, and high-contrast themes.**

> Remixed by **funkypitt** with **Claude Opus 4.6** from [OpenLibExtended](https://github.com/warreth/Openlib-Extended) by warreth and [OpenLib](https://github.com/dstark5/Openlib) by dstark5.

## Features

- **E-ink optimized reader** — paginated EPUB reader (epub.js) with adaptive font sizing, high-contrast dark/light themes, and tap zones designed for e-paper displays. Scroll view also available.
- **Library search** — search and browse books from online library catalogs. Filter by language, extension, and year.
- **Resilient downloads** — multi-instance engine automatically balances and fails over between mirrors.
- **Library management** — track your collection, discover trending books, and export files to other apps.
- **Cross-platform** — Android, iOS, Windows, Linux, and macOS.

## Installation

Grab the latest release from [GitHub Releases](https://github.com/funkypitt/funky-openlib/releases), or add the [F-Droid repo](https://funkypitt.github.io/fdroid-repo/repo).

```bash

flutter pub get

```
